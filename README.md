# Сборка для Vue 3

## Используемый стек
- Vue 3
- Typescript
- Vite
- TailwindCSS

## Плагины для Vue
- Pinia
- Vue Router
- Vue Final Modal
- Heroicons
- Tiny-Emitter

## Команды для сборки
Установить npm пакеты (требуется pnpm):
```shell
pnpm install --frozen-lockfile
```

Режим разработки (включенный vite сервер):
```shell
npm run dev
```

Сборка с обновлением ресурсов на лету:
```shell
npm run build-watch
```

Сборка в продакшн:
```shell
npm run build
```
